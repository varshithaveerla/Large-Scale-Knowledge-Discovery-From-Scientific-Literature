# Large-Scale Knowledge Discovery from the Enron Bag-of-Words Corpus

## Overview
This project implements an end-to-end distributed text analytics pipeline using PySpark to discover latent structure in large-scale textual data. The Enron Bag-of-Words email corpus is processed, analyzed, and modeled using scalable machine learning techniques.

## Components
- PySpark-based data ingestion and preprocessing
- Sparse vector feature engineering
- Unsupervised clustering (KMeans, LDA, Bisecting KMeans)
- Pseudo-labeled supervised classification
- Scalability and cost analysis
- Interactive Tableau dashboards

## Requirements
- Java 11
- Python 3.10+
- Apache Spark 3.5.8

## Execution
Use `run_pipeline.py` to execute the pipeline and `performance_profiler.py` to analyze runtime behavior.
